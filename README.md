# 🎂 Happy Birthday Quang Duy

Một trang web chúc mừng sinh nhật hiện đại, được thiết kế với tình yêu dành cho Quang Duy.

## ✨ Tính năng

- 🎵 **Nhạc Happy Birthday** được tạo bằng Web Audio API (không cần file nhạc)
- 🎆 **Hiệu ứng pháo hoa** và confetti động
- 🎂 **Bánh sinh nhật 3D** tương tác với chuột
- ✍️ **Hiệu ứng typewriter** cho lời chúc
- 🌌 **Particle system** nền động
- 📱 **Responsive** hoàn hảo trên mọi thiết bị
- 🎨 **Glassmorphism** + neon glow hiện đại
- ⌨️ **Phím tắt**: `M` (nhạc), `F` (pháo hoa), `C` (confetti)

## 🚀 Cách sử dụng

1. Mở file `index.html` trong trình duyệt hiện đại (Chrome, Firefox, Edge, Safari)
2. Click "Bật Nhạc" để nghe Happy Birthday
3. Click "Mở Quà" để xem điều bất ngờ
4. Click vào bánh sinh nhật để thổi nến!
